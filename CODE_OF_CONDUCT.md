# Code of Conduct

Nexa CRM contributors must communicate professionally and respectfully.

- Discuss code and decisions, not personal characteristics.
- Welcome questions and explain project conventions constructively.
- Give actionable review feedback and assume good intent.
- Do not harass, threaten, discriminate against or expose private information about others.
- Do not publish credentials, customer data or private security reports.

Maintainers may edit or remove contributions and restrict participation when behavior threatens contributors, users or the project. Conduct concerns should be reported privately to the maintainers through the repository's private security/contact facilities.
